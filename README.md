# ImpulseGen

Прошивка для STM32F446 — генератор сигналов произвольной формы.

## Что делает
- Генерация стандартных сигналов (синус, меандр, треугольник, пила) через 12-бит ЦАП + DMA
- Хранение пользовательских форм сигналов в EEPROM (I2C, 24Cxx)
- Управление через энкодер и кнопки: частота, амплитуда, смещение, выбор формы
- Отображение параметров на дисплее ST7789
- Связь с верхним уровнем по UART (бинарный протокол для чтения/записи записей)

## Стек
C, STM32 HAL, регистры напрямую (DAC, DMA, TIM, UART, GPIO), CMake

## Сборка
Проект собирается через CMake с пресетами из STM32CubeMX.
